拉取远程的Repo到本地（如果已经在本地，可以略过） 
$ git clone xxxxxx
在本地仓库删除文件 
$ git rm 我的文件
在本地仓库删除文件夹 
$ git rm -r 我的文件夹/
此处-r表示递归所有子目录，如果你要删除的，是空的文件夹，此处可以不用带上-r。
提交代码 
$ git commit -m"我的修改"
推送到远程仓库（比如GitHub） 
$ git push origin xxxxxx
